PALESTRANTE: André Barbosa. Formado em Sistemas de Informação pela EACH/USP desde 2017, trabalha desde então como cientista de dados, passando de startups até grandes empresas e desde Abril/2019 integra o time de data products do QuintoAndar. Apaixonado por construir soluções que impactem a vida das pessoas e que unem soluções da academia e engenharia, gosta do desafio de utilizar Machine Learning com dados reais para resolver problemas reais. Além disso, é professor do curso de Data Science & Machine Learning for Business da Tera.

DESCRIÇÃO: Machine Learning is shaping the industry, hands down. But a machine learning product is more than just fitting some model to some data. It involves having a multidisciplinary team to answer some HARD questions: what problem is there to solve? Is ML the only solution? Is there meaningful data to learn from? Which data transformations are required? How do we train this model? Is this model working? How do we provide predictions with this trained model? How do we scale it’s training? How do we scale it’s inference? How do we improve it in a agile way? This talk tries to show how hard this questions can be in a fast paced company, that grows 4x a year, and how QuintoAndar is working to reduce the complexity of taking experimentation to production.
