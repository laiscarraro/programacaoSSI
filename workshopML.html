TÍTULO: Workshop de machine learning

DESCRIÇÃO: a duração estimada do workshop é de 4 horas. Devido ao pouco tempo, todos os participantes precisam ter noção básica de python. Preferencialmente, vamos utilizar a plataforma google colab para o desenvolvimento, então acesso ao gmail e drive serão necessários. Os cases estarão pré prontos, apenas com algumas partes ‘faltando’ para discussão e realização prática.

CRONOGRAMA:

13h30min - 14h30min: Introdução + Overview ML (teórico) - Passaremos os principais tópicos de tarefas de aprendizado de máquina de maneira rápida (aquisição de dados, pré processamento, modelagem, métricas de resultados, deploy, etc.)
14h30min - 14h50min: Introdução ao Pandas (lib python) - Uma rápida introdução a lib que mais iremos usar durante o workshop.
14h50min - 15h30min: Case Pt. 1 - Descobrindo o que os dados têm para te falar - Faremos inicialmente um caso muito básico de Machine Learning, apresentando mais profundamente alguns tópicos passados na introdução.
15h30min - 17h10min: Case Pt. 2 - Torturando os dados até eles falarem - O segundo case será um pouco mais complexo, contando com a mesma estrutura do primeiro.
17h10min - 17h30min: Discussão Final + Apanhado geral + dúvidas.
