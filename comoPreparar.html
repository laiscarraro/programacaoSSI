PALESTRANTE: André Mariotti

DESCRIÇÃO: Conheça os desafios de implantar soluções de machine learning em organizações de grande porte e as soluções escaláveis e criativas para ter o seu negócio orientado a dados. Nesta conversa apresentaremos soluções para construcao de uma plataforma de ciência de dados que permite o trabalho colaborativo dos cientista de dados e também componentes projetados para servir esses modelos de forma produtiva e online. Além de abordas as soluções e tecnologias, serão apresentadas as 3 principais carreiras no mundo de dados, sendo elas a do Cientista de Dados, Engenheiro de Dados e Engenheiro de Machine Learning.
